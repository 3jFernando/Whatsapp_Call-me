Cómo usar Clic para chatear

La función Clic para chatear te permite comenzar un chat con alguien sin necesidad de tener su número de teléfono guardado en la libreta de contactos de tu teléfono. Si conoces el número de teléfono de una persona, puedes crear un enlace que te permitirá comenzar un chat con esa persona. Cuando hagas clic en el enlace, el chat con esa persona se abrirá. Clic para chatear funciona tanto en el teléfono como en WhatsApp Web.

Para crear tu propio enlace, utiliza https://wa.me/número luego reemplaza número con el número de teléfono completo en formato internacional. No incluyas los ceros, paréntesis o guiones cuando añadas el número de teléfono. Para aprender más sobre los números en formato internacional, lee este artículo. Por favor, recuerda que este número de teléfono debe tener una cuenta de WhatsApp activa.
